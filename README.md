# Expense-tracker
A feature-rich Expense Tracker web app with interactive charts, monthly analytics, dark/light mode, and persistent storage using JavaScript localStorage.

Expense Tracker Web App 💰

A simple and responsive Expense Tracker built using HTML, CSS, and JavaScript.

Features:
- Add income & expenses
- Categorize transactions
- Real-time balance calculation
- Local storage support
- Pie & monthly charts
- Dark/Light mode toggle

Tech Stack:
HTML, CSS, JavaScript

Purpose:
This project helps users track daily expenses and manage personal finance efficiently.
